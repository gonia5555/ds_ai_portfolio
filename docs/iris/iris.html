About the Data

The dataset contains information about three species of irises: Iris setosa, Iris versicolor, and Iris virginica.
The data includes measurements of four features: sepal length, sepal width, petal length, and petal width.
Each row in the dataset represents a single flower, with measurement values given in centimeters.
The dataset consists of 150 samples, with 50 samples for each species, and is widely used as a basic dataset for testing classification
algorithms and in data science and machine learning.
Columns:
  • sepal length - Sepal length in cm
  • sepal width - Sepal width in cm
  • petal length - Petal length in cm
  • petal width - Petal width in cm
  • class - Iris species (setosa, versicolor, virginica)

def classify_iris(petal_length, petal_width):
    if petal_length < 2.5:
        return "Iris-setosa"      # petal lenght < 2.5 cm  setosa 
    elif petal_width < 1.8:
        return "Iris-versicolor"   # If not Setosa, then consider the width < 1.8 cm  versicolor
    else:
        return "Iris-virginica"    # Other cases  virginica
